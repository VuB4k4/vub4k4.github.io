White Snake Stealer (Analysis)

Type: Infostealer / Malware-as-a-Service (MaaS)

Technical Overview

I conducted a static and dynamic analysis of this .NET-based malware found circulating on underground forums. It is designed to harvest sensitive information such as browser credentials, crypto wallets, and system data.

  • Obfuscation: Uses XOR encryption and dynamic API resolution to evade detection.
  • C2 Infrastructure: Connects to Telegram Bot API and uses Tor.
  • Persistence: Creates a Scheduled Task disguised as an "Nvidia" update.
